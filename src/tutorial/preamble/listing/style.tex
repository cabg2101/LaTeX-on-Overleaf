The \texttt{\textbackslash{}lstset} \gls{macro} defines the global typesetting \glspl{parameter} of the document. The \texttt{\textbackslash{}lstdefinelanguage} \gls{command} can be used to create our own programming language or, in this case, extend the definition of \LaTeX{} to allow syntax highlighting on more keywords. For more information on the \gls{package} and how to use it, feel free to consult the \gls{CTAN} documentation \parencite{web:ctan-listings}.
